Ключевые возможности

Расписание

Автоматическое переключение тем по заданному времени с поддержкой перехода через полночь

Мгновенное переключение

Быстрая смена тем Windows одним кликом из системного трея

Минимальная нагрузка

Работает в фоне с минимальным потреблением ресурсов (проверка раз в 5 секунд)

Автозапуск

Автоматический запуск с Windows, работа только в системном трее

Мультиязычность

Поддержка 5 языков: Русский, English, Français, Deutsch, Español

Мультимониторы

Поддержка любых конфигураций мониторов с диагностикой и синхронизацией

Как это работает

Контекстное меню в трее

  • Light theme / Темная тема: мгновенная смена темы
  • Расписание: включить/выключить автоматическое переключение
  • Время: настройка времени для светлой и темной темы
  • Автозапуск: запуск с Windows
  • Язык: выбор языка интерфейса

Системные требования

Минимальные требования

  • Windows 10/11 x64
  • .NET 8 (включен в инсталлятор)
  • Минимальное потребление RAM
  • Поддержка 1-10+ мониторов

Особенности установки

  • Self-contained сборка
  • Не требует отдельной установки .NET
  • Автоматическая настройка автозапуска
  • Полная совместимость с Explorer.exe
